# Synapse Digital Website

Modern multi-language IT services website built with Next.js and Tailwind CSS.

## Features

- ✅ Auto language detection (English, Hindi, German, Spanish)
- ✅ Responsive design
- ✅ Modern gradient UI
- ✅ Contact form
- ✅ Services showcase
- ✅ Mobile-friendly navigation

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide Icons

## Languages Supported

| Language | Code | Auto-detect |
|----------|------|-------------|
| English | en | ✅ |
| Hindi | hi | ✅ |
| German | de | ✅ |
| Spanish | es | ✅ |

## Local Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Open http://localhost:3000
```

## Build for Production

```bash
npm run build
```

Output in `dist/` folder.

## Deploy to Vercel

1. Push code to GitHub
2. Connect repo to Vercel
3. Deploy automatically

## Add More Languages

Edit `app/page.tsx` and add translations to the `translations` object.
